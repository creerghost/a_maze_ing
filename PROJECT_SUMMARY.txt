HOW TO USE
================================================================================

Installation:
  $ make install

Running:
  $ make run
  or
  $ python3 a_maze_ing.py config.txt

Building Package:
  $ python3 -m pip install build
  $ python3 -m build
  (Generates: mazegen-1.0.0-py3-none-any.whl)

Using as Library:
  from MazeGenerator import MazeGenerator, MazeSolver, MazeEngine
  
  config = {"WIDTH": "20", "HEIGHT": "15", ...}
  gen = MazeGenerator(config)
  gen.generate()
  solver = MazeSolver((0,0), (19,14))
  path = solver.solve(gen.grid)
